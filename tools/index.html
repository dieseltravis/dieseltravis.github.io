Tools

Tools Worthy Of Praise

Items strongly emphasized are newly discovered and/or especially great.

Web and Browser Extensions

  • jsbin.com - of all the online html/css/js editors, this is still my favorite
  • Slack - this is chat software we currently use at work
  • MightyText - keep your phone in your pocket and text/SMS from your computer
  • Feedly - a nice alternative to Google Reader
  • Stylish - override CSS in Firefox & Chrome (for example)

Cross-Platform

  • Synergy - share a keyboard, mouse, and clipboard across multiple computers on a network
  • f.lux - auto adjust monitor brightness based on daylight
  • namebench - benchmark your DNS and compare it to others to see which is best
  • OneNote - now the it's free for just about any OS I've been using it a lot more

Windows

  • Notepad2 - my favorite basic text editor
  • Paint .Net - my favorite text editor (be sure to get the optipng plugin)
    • where I work
  • Mailbird - this is currently my favorite email client (needs a calendar though)
  • Sharpkeys - I use this solely to map CAPSLOCK to Mute
  • WinMerge - compare files & folders
  • Pixie - find a color on the screen
  • Sysinternals - many windows utilities, autoruns and process explorer are 2 that I use the most (also a network share available to run them from directly)
  • ConEmu - tabbed console emulator that supports different shells
  • posh-git - this is the best way to use git on windows
  • WinDirStat - great for seeing what is taking up space on your drives
  • Virtual CloneDrive - a nice lightweight virtual disk drive
  • Expresso - handy tool for developing and debugging regular expressions
  • rooler - easily measure on-screen items
  • PhotoScape - an OK GIF editor
  • FileHippo Update Checker - scans for out of date stuff and provides links to updates

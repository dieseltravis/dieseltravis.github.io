Traviblog

Traviblog started out in 2002ish as an AIM Sub-Profile that I wrote using Classic ASP and Access. It was later migrated to a .Text blog which worked well for awhile. Then it was migrated to dasBlog which didn't work as well.

This site uses JSON for its data source to generate static files. I named the script suo because it was short, untaken, and latin is neat.
